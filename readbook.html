1. Introduction

Zota is a high-performance, memory-safe, statically typed programming language designed for stability and clarity.

2. File Structure & Commands

project/
 ├── src/
 │    └── main.zota
 ├── zota.toml

zota run main.zota
zota build main.zota

3. Libraries

add math;
add algorithm;
add io;

func main() {
    let nums: list<int> = [5,2,8,1];
    sort(nums);
    let result = sqrt(16);
    print(result);
}

No dot operator for library functions.

4. Variables & Types

let x: int = 10;
var score: int = 0;
const PI: float = 3.14;

Supported Types:

  • int
  • float
  • string
  • bool
  • char
  • list<T>
  • map<K,V>
  • tuple<T1,T2>

5. Functions

func add(a: int, b: int) -> int {
    let sum = a + b;
    return sum;
}

{ } mandatory for blocks. Semicolon only for multiple statements.

6. Control Flow

if (x > 10) {
    print("Big");
} else {
    print("Small");
}

for (let i: int = 0; i < 10; i++) {
    print(i);
}

7. Struct / Class / Interface

struct User {
    name: string
    age: int

    func greet() {
        print("Hello " + name);
    }
}

8. Error Handling

func divide(a: int, b: int) -> result<int,string> {
    if b == 0 { return error("Cannot divide by zero"); }
    return ok(a / b);
}

9. Ownership & Memory Safety

Each variable has a single owner. Moving a variable invalidates the previous owner. No garbage collector by default.
